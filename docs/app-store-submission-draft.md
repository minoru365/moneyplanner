# App Store 提出情報ドラフト

最終更新日: 2026-05-10

## 目的

App Store Connect で最終入力する前に、プライバシー栄養表示・公的データ出典・課金説明・返金/解約説明の文案を先に揃える。

## 1. プライバシー栄養表示（確認用ドラフト）

前提:

- 家計データは世帯共有機能の提供目的でのみ利用する
- 開発・分析・広告・外部提供への二次利用はしない
- 詳細は `docs/privacy-policy.md` を正とする

確認観点:

1. 収集するデータ型（例: ユーザー識別子、アプリ内生成データ）
2. 目的（アプリ機能提供、アカウント管理）
3. トラッキング有無（現行方針: トラッキング用途なし）
4. ユーザーに紐づくデータの扱い

注:

- 最終確定は App Store Connect の入力値で行う
- リリース時点の実装と矛盾がないか人間レビュー必須

## 2. 公的データ出典（表示文案）

ライフプランで用いる同梱データの出典候補:

- 家計調査（e-Stat, 00200561）
- 消費者物価指数 CPI（e-Stat, 00200573）
- 民間給与実態統計調査（e-Stat, 00351000）
- 毎月勤労統計（e-Stat, 00450071）
- 年金額（厚労省公表資料）

表示文案（アプリ説明/ポリシー向け）:

- 本アプリの一部シミュレーション初期値には、公的統計データ（e-Stat等）を加工して利用しています。出典情報はアプリ内およびサポート文書で確認できます。

## 3. 課金説明（文案テンプレート）

### A. 無料配布時

- 本バージョンでは追加課金はありません。

### B. 有料買い切り時

- アプリは買い切りで提供されます。追加課金やサブスクリプションはありません。

### C. サブスクリプション時

- 課金は Apple のサブスクリプションで管理されます。
- 購読期間、料金、更新条件は App Store 表示に従います。
- 解約は Apple ID のサブスクリプション管理画面から実施できます。

注:

- 現時点では価格モデル未確定のため、A/B/C から採用案を最終確定する。

## 4. 返金/解約説明（文案テンプレート）

- 解約や返金の可否・手続きは Apple のポリシーに従います。
- 返金申請は Apple の購入履歴画面から実施してください。
- サブスクリプション解約後の利用範囲（閲覧のみ/同期停止など）は、価格モデル確定時にアプリ内ヘルプへ明記します。

## 5. リリース前チェック

1. App Store Connect 入力値と `docs/privacy-policy.md` の整合
2. 公的データ出典IDと表示文言の整合
3. 価格モデル確定後の課金/解約文面更新
4. 問い合わせ導線（メール/URL）の最終反映
