# CSVインポートIAPは初期リリースでFamily Sharingを無効化（2026-07-11）

## 背景

CSVインポートは、非消耗型アプリ内課金 `csv_import_unlock`（¥300）で解放する。アプリの世帯は家計データを共有するための仕組みであり、AppleのFamily Sharingとは別の概念である。

## 決定

初期リリースでは、App Store ConnectのFamily Sharingを無効にする。CSVインポートのエンタイトルメントは、購入したApple IDに紐づける。

## 理由

世帯共有とAppleのFamily Sharingを連動させないことで、購入者に紐づく利用権を明確にする。これにより、世帯メンバーやApple家族グループの変更時に共有範囲が曖昧になること、ならびにサポートやエンタイトルメントに関する期待が不明確になることを避ける。

## 今後の見直し

Family Sharing対応は、商品ポリシーの見直しと実機での購入・復元・共有挙動の検証を行った場合にのみ再検討する。
